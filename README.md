# passgen
My favourite in-browser password generator is/was down, so I made this...

### Requirements
 - Python: 3.10
 - V: latest
### Python setup
```pip install ```
### V Build
```v passgen.v```
### Python Usage
 - `ln -s ./passgen.py $PATH/passgen # You get the idea`
 - `passgen`
 - or
 - `passgen -n 128`
### V Usage
 - `ln -s passgen $PATH/passgen`
 - `passgen`
 - or
 - `passgen --n 128`
